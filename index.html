This is an interactive version of the maps created by Aaron Schiff using census data describing population changes between 2001 and 2013. This visualization allows comparing between all ages category and young adults aged 20 to 34.

“In the following maps, there is one blue dot for each new person in census area units that experienced population growth over this time, and one red dot for each person lost in areas where the population shrank.”
